== Installation ==

1. Install as a regular WordPress plugin
2. You have to use shortcode for generate pro buttons  in post or page. Example shortcode is given below.
`[btn color="black" text="Your Text"][btn color="blue" text="Your Text"][btn color="green" text="Your Text"][btn color="orange"][btn color="purple"][btn color="pink"][btn color="navy"]` or `[btn_rounded color="black" text="Your Text"][btn_rounded color="blue" text="Your Text"]`
